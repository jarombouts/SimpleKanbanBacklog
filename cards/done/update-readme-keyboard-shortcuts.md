## Description

Update the README.md keyboard shortcuts section to include all the shortcuts
that have been added in recent iterations. The current documentation is missing
many shortcuts.

## Implementation Notes

Completed in iteration/017-update-readme-shortcuts branch:
- Reorganized keyboard shortcuts into categories
- Added all missing shortcuts:
  - Home/End for first/last card
  - Cmd+Up/Down for reordering
  - Cmd+Left/Right for moving between columns
  - Cmd+A for select all in column
  - Cmd+D for duplicate
  - Cmd+F for search
  - Cmd+Shift+N for new card
  - Cmd+Z/Shift+Z for undo/redo
- Categories: Navigation, Card Actions, Moving Cards, Selection & Search, Window Management
